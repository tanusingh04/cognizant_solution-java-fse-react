Debugging Practice with Chrome DevTools

This page is designed to help you practice using Chrome DevTools (Inspect, Console, and Sources panel for Breakpoints).

Instructions:

  1. Right-click anywhere and select Inspect to open DevTools.
  2. Go to the Console tab to view output logs from our script.
  3. Go to the Sources tab, open this HTML file, find the calculateTotal() function, and click on the line number inside it to set a breakpoint.
  4. Click the "Calculate Registration Cost" button to trigger the breakpoint and watch the variable values change.
