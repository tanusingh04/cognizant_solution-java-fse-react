This page links an external stylesheet (Problem_01_styles.css) and sets up a mock layout box for styling experiments.

Inspect Me and Change My Styles!

How to Complete the Exercise Tasks:

  • Task 1: Simulate Different Screen Sizes
    Press F12 or right-click → Inspect. Click the Toggle Device Toolbar icon (smartphone/tablet icon in top-left of DevTools) to simulate various screen widths (e.g., iPhone, iPad, Desktop).
  • Task 2: Inspect and Test Styles Live
    Select the pointer tool in DevTools, click on the blue box above (.box-demo). In the Styles panel on the right, change background-color to another color (like #ef4444) or add transform: rotate(5deg) to see changes instantly.
  • Task 3: Check External CSS Loading
    Go to the Network tab in DevTools. Reload the page (Cmd + R or Ctrl + R). Look for Problem_01_styles.css in the list of loaded assets to confirm it loaded with a 200 OK status.
