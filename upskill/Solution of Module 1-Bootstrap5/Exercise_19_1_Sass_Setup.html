Bootstrap Customization with Sass

Steps to set up Sass Customization:

  1. Initialize npm: npm init -y
  2. Install Bootstrap and Sass compiler: npm install bootstrap sass
  3. Create a stylesheet named custom.scss.
  4. Import Bootstrap inside your scss: 
    @import "../node_modules/bootstrap/scss/bootstrap";
  5. Compile the stylesheet: npx sass custom.scss custom.css
