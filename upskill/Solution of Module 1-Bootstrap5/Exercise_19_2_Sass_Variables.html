Sass Variables Customization

To customize Bootstrap variables like primary colors or border radiuses, modify them before importing Bootstrap in your SCSS file:

// 1. Define custom overrides first
$primary: #e066ff;
$border-radius: 10px;

// 2. Import Bootstrap SCSS
@import "../node_modules/bootstrap/scss/bootstrap";

Compile the stylesheet using the compiler script to apply these changes to the compiled CSS bundle.
